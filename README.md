# Banking Management System (Frontend Demo)

This project demonstrates the frontend user interface and workflow of a Banking Management System.

## Features
- User Login (UI only)
- Account Dashboard
- Transaction History
- Fund Transfer Interface

## Technologies Used
- HTML
- CSS
- JavaScript

## Live Demo
https://haleemdevop.github.io/banking-management-system/

## Note
This project showcases the frontend functionality. Backend services were implemented separately using Spring Boot and MySQL and tested locally.
